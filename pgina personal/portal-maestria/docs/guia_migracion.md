# Guía de Migración y Continuidad con Antigravity

Esta guía ha sido creada para que puedas mover tu proyecto a otra computadora y seguir trabajando con tu asistente de IA sin perder el hilo.

## 1. Sincronización del Código
Como hemos configurado todo con **rutas relativas**, solo necesitas clonar el repositorio o copiar la carpeta:

- **Repositorio**: `https://github.com/massoni007/web-pagina-maestria.git`
- **Inicio rápido**: Ejecuta `start_portal.bat` en la nueva computadora.

## 2. Continuidad de la IA (Antigravity)
- **Historia de Chat**: Si accedes con la misma cuenta y abres esta conversación, verás todo nuestro historial de mensajes.
- **Contexto del Proyecto**: Aunque el chat se pierda, yo puedo leer este archivo y los documentos en la carpeta `/docs` para entender exactamente en qué nos quedamos.

## 3. Archivos Clave
- `portal-maestria/start_portal.bat`: Arranca el servidor local.
- `portal-maestria/docs/`: Contiene esta guía y el resumen de cambios.

---
*Nota: He movido esta guía dentro de tu carpeta de proyecto para asegurarme de que viaje contigo a través de GitHub o USB.*
